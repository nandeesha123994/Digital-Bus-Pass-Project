How to Add Rewards Buttons to Your Dashboards

Step 1: Create the includes directory

Create a directory named includes in your project root if it doesn't exist already.

Step 2: Create rewards-nav.php

Create a new file named rewards-nav.php in the includes directory with this content:

<?php
// Get user's reward points if logged in
$userPoints = 0;
$userBadge = "Bronze";
if (isset($_SESSION["uid"])) {
    $pointsQuery = "SELECT reward_points FROM users WHERE id = ?";
    $stmt = $con->prepare($pointsQuery);
    $stmt->bind_param("i", $_SESSION["uid"]);
    $stmt->execute();
    $result = $stmt->get_result();
    if ($row = $result->fetch_assoc()) {
        $userPoints = $row["reward_points"];
        // Determine badge based on points
        if ($userPoints >= 1000) {
            $userBadge = "Diamond";
        } elseif ($userPoints >= 500) {
            $userBadge = "Platinum";
        } elseif ($userPoints >= 250) {
            $userBadge = "Gold";
        } elseif ($userPoints >= 100) {
            $userBadge = "Silver";
        }
    }
}
?>

<!-- User Dashboard Navigation -->
<?php if (isset($_SESSION["uid"])): ?>
<div class="nav-item rewards-nav">
    <a href="rewards-dashboard.php" class="rewards-link">
        <i class="fas fa-star"></i>
        <span class="rewards-text">My Rewards</span>
        <span class="rewards-badge <?php echo strtolower($userBadge); ?>">
            <?php echo $userPoints; ?> pts
        </span>
    </a>
</div>
<?php endif; ?>

<!-- Admin Dashboard Navigation -->
<?php if (isset($_SESSION["admin_logged_in"]) && $_SESSION["admin_logged_in"] === true): ?>
<div class="nav-item rewards-nav">
    <a href="admin-rewards.php" class="rewards-link">
        <i class="fas fa-star"></i>
        <span class="rewards-text">Rewards Management</span>
    </a>
</div>
<?php endif; ?>

Step 3: Add to user-dashboard.php

Find this line in your user-dashboard.php:

<a href="apply-pass.php" class="nav-item">
    <i class="fas fa-plus-circle"></i> Apply for Pass
</a>

Add this line right after it:

<?php include("includes/rewards-nav.php"); ?>

Step 4: Add to admin-dashboard.php

Find this line in your admin-dashboard.php:

<a href="manage-applications.php" class="nav-item">
    <i class="fas fa-file-alt"></i> Applications
</a>

Add this line right after it:

<?php include("includes/rewards-nav.php"); ?>

Important Notes:

  • Make sure you have Font Awesome included in your project for the icons
  • The rewards navigation will only show up for logged-in users
  • Admin users will see the "Rewards Management" link
  • Regular users will see their points and badge level

Step 5: Test the Changes

After making these changes:

  1. Log in as a regular user to see the "My Rewards" button with points
  2. Log in as an admin to see the "Rewards Management" button
  3. Make sure the styling looks correct in both cases
